University Student Management System

This project is an application built with Java, JDBC, and PostgreSQL to manage university data. It allows users to add, view, update, and delete students and courses, enroll students in courses, and display enrolled courses using SQL JOIN queries. The project applies Object-Oriented Programming principles, Java Collections, and the DAO design pattern to create a clean and organized system.

 Features
- Manage students and courses (CRUD operations)
- Enroll students in courses
- View student enrollments using SQL JOIN queries
- Connect Java applications to PostgreSQL using JDBC
- Apply OOP concepts and the DAO design pattern
